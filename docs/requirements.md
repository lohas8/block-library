# 社区图书管理系统 - 需求文档

## 1. 项目概述

### 1.1 项目背景
- **项目名称**：社区共享图书管理系统
- **目标用户**：小区/乡村居民、业委会
- **核心价值**：促进邻里共享阅读，打造社区文化氛围

### 1.2 系统规模
- **多用户**：社区居民（业主）
- **图书来源**：业主共享 + 业委会采购

---

## 2. 功能需求

### 2.1 图书管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| ISBN录入 | 扫描ISBN自动获取图书信息（书名、作者、出版社、封面） | P0 |
| 手动添加 | 手动输入图书信息 | P0 |
| 图书分类 | 按类型分类（文学/科技/儿童/历史等） | P0 |
| 图书标签 | 自定义标签（热门/推荐/新书） | P1 |
| 图书状态 | 在馆/借出/预约/下架 | P0 |

### 2.2 借阅管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 共享图书 | 业主贡献自己的书籍，设置可借阅状态 | P0 |
| 借书 | 借阅图书，记录借阅时间 | P0 |
| 还书 | 归还图书，自动计算是否逾期 | P0 |
| 预约 | 预约已借出的图书 | P1 |
| 借阅规则 | 设置每人限借数量、借阅时长上限 | P0 |

### 2.3 读者管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 注册/登录 | 手机号注册，验证码登录 | P0 |
| 个人信息 | 头像、昵称、联系方式 | P0 |
| 会员等级 | 普通会员/VIP会员（业委会指定） | P1 |
| 借阅记录 | 查看个人借阅历史 | P0 |

### 2.4 积分系统 ⭐
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 获得积分 | 共享图书 +20分 / 每次借阅 +5分 / 归还及时 +10分 | P0 |
| 积分规则 | 自定义积分规则（管理员配置） | P1 |
| 积分兑换 | 积分兑换小礼品/优先借阅权 | P0 |
| 积分记录 | 查看积分收支明细 | P0 |

### 2.5 通知提醒
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 到期提醒 | 还书前3天推送提醒 | P0 |
| 逾期通知 | 逾期未还提醒 | P0 |
| 预约到书 | 预约的图书可借提醒 | P1 |
| 新书上架 | 系统通知/推送 | P2 |

### 2.6 数据统计
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 借阅排行 | 热门图书TOP10 | P1 |
| 阅读统计 | 个人借阅数量、阅读时长 | P1 |
| 库存统计 | 在馆/借出/总藏书量 | P1 |
| 积分排行 | 积分排行榜 | P1 |

### 2.7 数据管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 导入数据 | CSV/Excel导入图书、读者数据 | P1 |
| 导出数据 | 导出借阅记录、图书列表 | P1 |

---

## 3. 用户角色

| 角色 | 权限 |
|------|------|
| **普通用户** | 注册、浏览图书、借阅、还书、共享图书、积分兑换 |
| **图书管理员** | 图书管理、读者管理、借阅审核、分类管理 |
| **系统管理员** | 所有权限 + 用户管理、系统设置、数据管理 |

---

## 4. 权限模块

### 4.1 角色定义
```javascript
// 角色枚举
Role: {
  USER: 'user',           // 普通用户
  LIBRARIAN: 'librarian', // 图书管理员
  ADMIN: 'admin'          // 系统管理员
}

// 权限枚举
Permission: {
  // 图书相关
  'book:create',    // 添加图书
  'book:read',      // 查看图书
  'book:update',   // 修改图书
  'book:delete',   // 删除图书
  
  // 读者相关
  'user:create',   // 创建用户
  'user:read',      // 查看用户
  'user:update',    // 修改用户
  'user:delete',   // 删除用户
  
  // 借阅相关
  'borrow:create', // 创建借阅
  'borrow:approve', // 审批借阅
  'borrow:return', // 办理还书
  
  // 系统相关
  'system:config', // 系统配置
  'data:export',    // 导出数据
  'data:import'     // 导入数据
}
```

|------|:--------:|:----------:|:----------:|
| 注册/登录 | ✅ | ✅ | ✅ |
| 浏览图书 | ✅ | ✅ | ✅ |
| 借阅图书 | ✅ | ✅ | ✅ |
| 共享图书 | ✅ | ✅ | ✅ |
| 积分兑换 | ✅ | ✅ | ✅ |
| 图书管理 | ❌ | ✅ | ✅ |
| 借阅审批 | ❌ | ✅ | ✅ |
| 读者管理 | ❌ | ✅ | ✅ |
| 系统设置 | ❌ | ❌ | ✅ |
| 数据导入/导出 | ❌ | ✅ | ✅ |

### 4.3 权限验证方式
- **前端**：路由守卫 + 按钮级别权限控制
- **后端**：中间件验证（API级别权限控制）

## 5. 非功能性需求

- **性能**：支持100+用户同时在线
- **安全**：密码加密存储、登录验证
- **易用**：简洁清晰的界面设计
- **可维护**：代码规范、注释完整
